It is a simple Go REST API application built to learn Go based on the [Go - The Complete Guide](https://www.udemy.com/course/go-the-complete-guide) course.

At the end of the course, this app was created. We created an event organizer REST API service that includes:

- Fetching existing events
- Creating, editing, and deleting events by authorized users
- Simple JWT authentication
- Registration and cancellation for events

With this course, I learned the fundamentals of Go, generics, goroutines, Gin, and debugging with Delve (which is not covered in the course).
